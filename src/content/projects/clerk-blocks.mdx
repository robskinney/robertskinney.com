This project was started with a goal of allowing developers to quickly clone commonly-used authentication components into their application using a shadcn/ui registry.

## Components

Components are separated by 5 sections. These are comprised of User, User Settings, Organization, and Organization Settings components.

### Authentication

#### Sign In

This component is configurable to allow users to sign in via OAuth providers or by email/username and password.

![Screenshot of the sign-in-1 component.](/images/clerk-blocks/authentication/sign-in.png)

#### Sign Up

This component is configurable to allow users to sign up using OAuth providers or by using their name, email/username, and password.

![Screenshot of the sign-up-1 component.](/images/clerk-blocks/authentication/sign-up.png)

### User

#### User Dropdown 1

This component is a simple user dropdown that is best used in a navigation bar.

![Screenshot of the user-dropdown-1 component.](/images/clerk-blocks/user/user-dropdown-1.png)

#### User Dropdown 2

This component is a simple user dropdown that is best used in a sidebar.

![Screenshot of the user-dropdown-2 component.](/images/clerk-blocks/user/user-dropdown-2.png)

### User Settings

#### Email Management

This component allows the user to add and remove emails, send verification emails, and set their primary email address.

![Screenshot of the email-management-1 component.](/images/clerk-blocks/user-settings/email-management.png)

#### Profile Form

This component allows the user to adjust their profile picture as well as their name, first name, and last name.

![Screenshot of the profile-form-1 component.](/images/clerk-blocks/user-settings/profile-form.png)

#### Session Management

This component allows the user to view and log out their active sessions across devices.

![Screenshot of the session-management-1 component.](/images/clerk-blocks/user-settings/session-management.png)

#### Social Accounts

This component allows the user to link and unlink social accounts across OAuth providers.

![Screenshot of the social-accounts-1 component.](/images/clerk-blocks/user-settings/social-accounts.png)

### Organization

This component is a simple organization switcher that is best used in a sidebar.

This also allows the user to create new organizations.

![Screenshot of the organization-switcher-1 component.](/images/clerk-blocks/organization/organization-switcher.png)

### Organization Settings

#### Organization Form

This component allows an organization user to adjust the organization's profile picture as well as its name.

![Screenshot of the organization-form-1 component.](/images/clerk-blocks/organization-settings/organization-form.png)
